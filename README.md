# IOPaint API 测试工具

用于对接 IOPaint API 接口，支持图片和视频去水印功能。

## 功能特点

- ✅ 支持图片去水印
- ✅ 支持视频去水印（逐帧处理）
- ✅ 支持自定义水印区域
- ✅ 支持提示词引导修复
- ✅ 进度条显示处理状态
- ✅ 图形界面操作（GUI 版本）
- ✅ 命令行操作（CLI 版本）

## 环境要求

- Python 3.7+
- 依赖包：
  - requests
  - opencv-python
  - numpy
  - tqdm
  - Pillow（GUI 版本需要）

## 安装依赖

```bash
pip install requests opencv-python numpy tqdm Pillow
```

## 使用方法

### 1. 图形界面版本（推荐）

```bash
python iopaint_gui.py
```

**操作步骤：**
1. 点击「选择文件」按钮选择要处理的图片或视频
2. 系统会自动生成输出路径，也可以点击「选择输出」按钮自定义
3. 在水印区域部分输入 X、Y 坐标和宽高，点击「添加区域」
4. 可以添加多个水印区域
5. 输入提示词（可选）
6. 点击「开始处理」按钮
7. 等待处理完成，查看处理结果

### 2. 命令行版本

```bash
python iopaint_api.py
```

然后选择选项 1（处理图片）或 2（处理视频），输入：
- 文件路径
- 输出路径
- 水印区域（格式：x y width height）
- 提示词（可选）

## API 接口

- 接口地址: `http://8.149.128.46:7860/api/v1/inpaint`
- 请求方法: POST
- 请求格式: JSON
- 请求参数:
  - `image`: Base64 编码的图片数据
  - `mask`: Base64 编码的掩码数据
  - `prompt`: 提示词（可选）

## 注意事项

1. 视频处理会逐帧进行，处理时间较长，请耐心等待
2. 建议使用 GPU 版本的 IOPaint 服务以获得更好的性能
3. 对于大视频，可能需要增加超时时间
4. 水印区域坐标是相对于图片/视频分辨率的像素值
5. 图形界面版本需要安装 Pillow 库

## 故障排查

- 如果 API 调用失败，请检查网络连接和 API 服务是否正常运行
- 如果视频处理失败，可能是因为内存不足，建议处理较短的视频
- 如果图片处理效果不理想，可以尝试调整提示词或水印区域
- 如果图形界面无法启动，请检查是否安装了 Pillow 库

## 配置文件

可以通过修改 `config.py` 文件来调整 API 配置：

```python
API_CONFIG = {
    "iopaint": {
        "enabled": True,
        "api_url": "http://8.149.128.46:7860/api/v1/inpaint",
        "timeout": 300  # 5分钟超时
    }
}
```
